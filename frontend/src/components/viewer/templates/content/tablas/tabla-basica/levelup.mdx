# Tabla básica con react-pdf-levelup

Una tabla simple de 4 columnas usando los componentes semánticos
`Table` / `Thead` / `Tbody` / `Tr` / `Th` / `Td` de `@react-pdf-levelup/core`.
No hace falta escribir `View`, `Text` ni `StyleSheet` a mano: el componente
`Table` ya resuelve bordes, encabezado y alineación por columna.

```tsx
import { Layout, H1, P, Table, Thead, Tbody, Tr, Th, Td } from '@react-pdf-levelup/core'

const rows = [
  ["Ada Lovelace", "Ingeniería", "Senior", "$4,200"],
  ["Alan Turing", "Investigación", "Principal", "$5,100"],
  ["Grace Hopper", "Plataforma", "Staff", "$4,800"],
  ["Margaret Hamilton", "Calidad", "Senior", "$4,300"],
]

const TablaBasica = () => (
  <Layout size="A4" meta={{ title: "Tabla básica", language: "es" }}>
    <H1>Equipo de Ingeniería</H1>
    <P>Listado simple de integrantes del equipo con su rol y salario base.</P>

    <Table cellHeight={26} headerBackground="#1d4ed8" textColor="#111827">
      <Thead textColor="#ffffff">
        <Tr>
          <Th>Nombre</Th>
          <Th>Área</Th>
          <Th>Nivel</Th>
          <Th textAlign="right">Salario</Th>
        </Tr>
      </Thead>
      <Tbody>
        {rows.map((r, i) => (
          <Tr key={i}>
            <Td>{r[0]}</Td>
            <Td>{r[1]}</Td>
            <Td>{r[2]}</Td>
            <Td textAlign="right">{r[3]}</Td>
          </Tr>
        ))}
      </Tbody>
    </Table>
  </Layout>
)

export default TablaBasica
```

## Notas

- `Tr` **debe** ser hijo directo de `Thead`/`Tbody`, y `Th`/`Td` hijos directos
  de `Tr`. El componente inyecta el índice de fila y el ancho por `colSpan`
  vía `cloneElement` sobre hijos inmediatos; envolver una fila en un
  componente auxiliar rompe ese cálculo silenciosamente.
- `headerBackground` y `textColor` en `Thead` sobreescriben los valores
  heredados de `Table` solo para el encabezado.
- No es necesario calcular anchos de columna a mano: `Tr` los reparte de
  forma proporcional según `colSpan` de cada celda.
