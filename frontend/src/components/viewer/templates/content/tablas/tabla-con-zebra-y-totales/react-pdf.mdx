# Tabla con zebra y totales con @react-pdf/renderer

Misma factura, construida a mano con `View`/`Text`/`StyleSheet`. El
alternado de color (zebra) y la fusión de la fila de totales se resuelven
calculando índices y anchos manualmente.

```tsx
import { Document, Page, View, Text, StyleSheet } from '@react-pdf/renderer'

const items = [
  ["Licencia software anual", 2, "$1,200.00", "$2,400.00"],
  ["Soporte técnico premium", 1, "$800.00", "$800.00"],
  ["Hosting y dominio", 12, "$45.00", "$540.00"],
  ["Consultoría de migración", 8, "$150.00", "$1,200.00"],
  ["Capacitación al equipo", 3, "$300.00", "$900.00"],
]

const total = items.reduce(
  (acc, r) => acc + parseFloat(String(r[3]).replace(/[$,]/g, "")),
  0
)

const styles = StyleSheet.create({
  page: { padding: 30, fontSize: 10 },
  h1: { fontSize: 20, fontWeight: 700, marginBottom: 8 },
  p: { color: "#374151", marginBottom: 16 },
  headerRow: { flexDirection: "row", backgroundColor: "#111827" },
  headerCell: { flex: 1, padding: 6, color: "#ffffff", fontWeight: 700 },
  row: { flexDirection: "row", borderBottomWidth: 1, borderColor: "#cbd5e1" },
  rowZebra: { backgroundColor: "#f1f5f9" },
  cell: { flex: 1, padding: 6 },
  cellRight: { textAlign: "right" },
  totalRow: { flexDirection: "row", borderBottomWidth: 1, borderColor: "#cbd5e1" },
  totalLabelCell: { flex: 3, padding: 6, textAlign: "right", fontWeight: 700 },
  totalValueCell: { flex: 1, padding: 6, textAlign: "right", fontWeight: 700 },
})

const columns = ["Concepto", "Cant.", "Precio unit.", "Subtotal"]

const TablaConTotales = () => (
  <Document>
    <Page size="A4" style={styles.page}>
      <Text style={styles.h1}>Factura de Servicios</Text>
      <Text style={styles.p}>
        Detalle de conceptos facturados durante el trimestre, con fila de
        totales al final.
      </Text>

      <View>
        <View style={styles.headerRow}>
          {columns.map((col, i) => (
            <Text
              key={col}
              style={[styles.headerCell, i > 0 && styles.cellRight]}
            >
              {col}
            </Text>
          ))}
        </View>

        {items.map((row, i) => (
          <View
            key={i}
            // El zebra hay que calcularlo a mano según la paridad del índice:
            // react-pdf-levelup hace exactamente esto internamente en `Tbody`.
            style={[styles.row, i % 2 === 1 && styles.rowZebra]}
          >
            {row.map((value, j) => (
              <Text
                key={j}
                style={[styles.cell, j > 0 && styles.cellRight]}
              >
                {value}
              </Text>
            ))}
          </View>
        ))}

        {/* Fila de totales: la fusión de columnas se simula con flex:3 en
            vez de un colSpan real, ya que @react-pdf/renderer no tiene
            noción de "fusionar celdas" fuera de la capa de react-pdf-levelup. */}
        <View style={styles.totalRow}>
          <Text style={styles.totalLabelCell}>Total</Text>
          <Text style={styles.totalValueCell}>${total.toFixed(2)}</Text>
        </View>
      </View>
    </Page>
  </Document>
)

export default TablaConTotales
```

## Notas

- No existe `colSpan` en los primitivos base: fusionar celdas se simula
  dándole a la celda "fusionada" un `flex` equivalente a la suma de las
  columnas que ocuparía (`flex: 3` para 3 de las 4 columnas).
- El alternado zebra se calcula a mano con `i % 2`, mientras que en
  react-pdf-levelup lo resuelve automáticamente el componente `Tbody` a
  partir de la prop `zebraColor` de `Table`.
