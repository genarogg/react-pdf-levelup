## Left

Aligns content to the left.

### Props

| Name    | Type     | Default | Description           |
| :------ | :------- | :------ | :-------------------- |
| `style` | `object` | `{}`    | Additional styles.    |

### Example

```tsx
<Left><P>Text to the left</P></Left>
```
