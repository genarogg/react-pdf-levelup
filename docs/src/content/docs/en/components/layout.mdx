# Layout

The **Layout** component is the structural core of the PDF document.

Internally, it wraps the content with `<Document>` and `<Page>` from `@react-pdf/renderer` and automatically manages:

- Page size
- Orientation
- Predefined margin system
- Background color or image
- Dynamic footer
- Automatic numbering
- Centimeter reference grid
- Debug mode
- Automatic footer space reservation

# Margin System

The Layout includes a predefined margin system to facilitate layout.

## `margin="normal"`

Uses the `padding` value.

```tsx
<Layout padding={40} margin="normal" />
```

## `margin="estrecho"`

Applies 36pt on all sides.

## `margin="ancho"`

Applies 108pt on all sides.

## `margin="apa"`

Applies 72pt (1 inch) on all sides.

Ideal for academic documents.

# NextPage

The **NextPage** component allows you to insert a manual page break within the document. It internally uses `<View break />` from `@react-pdf/renderer` to force the start of a new page.

## Usage

```tsx
<Layout>
  <H1>First Page</H1>
  <P>Content of the first page...</P>
  
  <NextPage />
  
  <H1>Second Page</H1>
  <P>Content of the second page...</P>
</Layout>
```

## Use Cases

- Separate important sections of the document
- Start chapters or sections on a new page
- Manually control content distribution
- Prevent important content from splitting across pages

# Complete Example

```tsx
<Layout
  size="A4"
  orientation="portrait"
  margin="apa"
  backgroundColor="#ffffff"
  backgroundImage="/marca-agua.png"
  backgroundImageOpacity={0.15}
  footer={<P>My Company</P>}
  lines={2}
  pagination
  rule={false}
  style={{ fontFamily: 'Helvetica' }}
>
  <H1>Document Title</H1>
  <P>Document content...</P>
</Layout>
```
