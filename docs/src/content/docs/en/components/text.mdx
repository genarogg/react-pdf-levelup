## P, H1-H6

Basic text components (paragraph and headings). They support all properties of the `Text` component from `@react-pdf/renderer`.

### Props

| Name    | Type     | Default | Description           |
| :------ | :------- | :------ | :-------------------- |
| `style` | `object` | `{}` | Additional text styles. |
| `debug` | `boolean` | `false` | Enables debug mode (borders). |
| `fixed` | `boolean` | `false` | Indicates if the element should appear on all pages. |
| `break` | `boolean` | `false` | Forces a page break before the element. |

### Example

```tsx
import { H1, H3, P } from '@react-pdf-levelup/core';

<H1>Title</H1>
<H3>Subtitle</H3>
<P>Paragraph with normal text.</P>
```

## Strong, Em, U, Small

Inline text formatting components. They support all properties of the `Text` component.

- **Strong**: Bold text.
- **Em**: Italic text (emphasis).
- **U**: Underlined text.
- **Small**: Smaller text.

### Props

| Name    | Type     | Default | Description           |
| :------ | :------- | :------ | :-------------------- |
| `style` | `object` | `{}` | Additional styles. |
| `debug` | `boolean` | `false` | Enables debug mode (borders). |
| `fixed` | `boolean` | `false` | Indicates if the element should appear on all pages. |
| `break` | `boolean` | `false` | Forces a page break before the element. |

### Example

```tsx
import { P, Strong, Em, U, Small } from '@react-pdf-levelup/core';

<P>
  <Strong>Bold</Strong>, <Em>Italic</Em>, <U>Underlined</U>, <Small>Small</Small>
</P>
```
