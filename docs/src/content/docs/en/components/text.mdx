## P, H1-H6

Basic text components (paragraph and headings).

### Props

| Name    | Type     | Default | Description           |
| :------ | :------- | :------ | :-------------------- |
| `style` | `object` | `{}`    | Additional text styles. |

### Example

```tsx
<H1>Title</H1>
<H3>Subtitle</H3>
<P>Paragraph with normal text.</P>
```

## Strong, Em, U, Small

Inline text formatting components.

- **Strong**: Bold text.
- **Em**: Italic text (emphasis).
- **U**: Underlined text.
- **Small**: Smaller text.

### Props

| Name    | Type     | Default | Description           |
| :------ | :------- | :------ | :-------------------- |
| `style` | `object` | `{}`    | Additional styles.    |

### Example

```tsx
<P>
  <Strong>Bold</Strong>, <Em>Italic</Em>, <U>Underlined</U>, <Small>Small</Small>
</P>
```
