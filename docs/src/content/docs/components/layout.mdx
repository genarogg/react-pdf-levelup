## LayoutPDF

Componente principal para configurar el documento PDF. Envuelve todo tu contenido.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `size` | `string` | `"A4"` | Tamaño de la página (A0-A9, Letter, Legal, Tabloid). |
| `orientation` | `string` | `"vertical"` | Orientación de la página (vertical, horizontal). |
| `backgroundColor` | `string` | `"white"` | Color de fondo de la página. |
| `padding` | `number` | `30` | Padding interno de la página en puntos (solo si margen=normal). |
| `margen` | `string` | `"normal"` | Tipo de margen (apa, normal, estrecho, ancho). |
| `style` | `object` | `{}` | Estilos adicionales para la página. |
| `footer` | `ReactNode` | `""` | Contenido del pie de página. |
| `lines` | `number` | `1` | Número de líneas para el pie de página (1-10). |
| `pagination` | `boolean` | `true` | Mostrar numeración de páginas. |
| `rule` | `boolean` | `false` | Mostrar rejilla de referencia en la página. |

### Ejemplo

```tsx
<LayoutPDF pagination rule footer={<P>Pie de página</P>} lines={2}>
  <H1>Título</H1>
  <P>Contenido de ejemplo</P>
</LayoutPDF>
```

## Container

Contenedor principal con padding horizontal. Útil para centrar contenido o darle márgenes consistentes.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales para el contenedor. |

### Ejemplo

```tsx
<Container>
  <Row>
    <Col6><P>Columna 1</P></Col6>
    <Col6><P>Columna 2</P></Col6>
  </Row>
</Container>
```

## Row

Fila para el sistema de grid. Se utiliza para agrupar columnas.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales para la fila. |

### Ejemplo

```tsx
<Row>
  <Col4><P>A</P></Col4>
  <Col4><P>B</P></Col4>
  <Col4><P>C</P></Col4>
</Row>
```

## Col1-Col12

Columnas para el sistema de grid (de 1 a 12 unidades). El sistema de grid se basa en 12 columnas.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales para la columna. |

### Ejemplo

```tsx
<Col12><P>Contenido a ancho completo</P></Col12>
```

## Div

Contenedor genérico para agrupar elementos. Similar al `div` de HTML.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales. |

### Ejemplo

```tsx
<Div style={{ padding: 10 }}>
  <P>Bloque con padding</P>
</Div>
```
