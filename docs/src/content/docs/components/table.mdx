## Table (Tablet)

Componente principal de tabla.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales para la tabla. |

### Ejemplo

```tsx
<Table>
  <Thead>
    <Tr>
      <Th>Col A</Th>
      <Th textAlign="right">Col B</Th>
    </Tr>
  </Thead>
  <Tbody>
    <Tr>
      <Td>1</Td>
      <Td>2</Td>
    </Tr>
    <Tr>
      <Td>3</Td>
      <Td>4</Td>
    </Tr>
  </Tbody>
</Table>
```

## Thead

### Props

| Nombre | Tipo | Default | Descripción |

| `style` | `object` | `{}` | Estilos adicionales. |

### Ejemplo

```tsx
<Thead><Tr><Th>Columna</Th></Tr></Thead>
```

## Th

Celda de encabezado (`TableHeader`).

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales. |
| `width` | `string\|number` | `""` | Ancho de la celda. |
| `height` | `string\|number` | `""` | Alto de la celda. |
| `cellSize` | `string` | `"medium"` | Tamaño predefinido (small, medium, large). |
| `colSpan` | `number` | `""` | Número de columnas que ocupa. |
| `textAlign` | `string` | `"left"` | Alineación del texto (left, center, right). |

### Ejemplo

```tsx
<Th textAlign="center">Encabezado</Th>
```
