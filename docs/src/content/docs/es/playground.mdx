El **Playground** es el corazón de React PDF Level Up. Es un entorno de desarrollo interactivo que te permite escribir código React PDF y ver los resultados en tiempo real.

## Características Principales

<CardGrid>
  <Card title="Editor en Tiempo Real" icon="pencil">
    Escribe código TypeScript/React a la izquierda y ve el PDF renderizado
    instantáneamente a la derecha.
  </Card>
  <Card title="Persistencia Automática" icon="seti:db">

  <Card title="Templates Predefinidos" icon="document">

    empezar rápidamente.
  </Card>
</CardGrid>

El editor soporta atajos de teclado estándar de editores de código (tipo VS Code) para facilitar la edición.
