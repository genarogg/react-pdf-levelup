## P, H1-H6

Componentes básicos de texto (párrafo y encabezados). Soportan todas las propiedades del componente `Text` de `@react-pdf/renderer`.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales para el texto. |
| `debug` | `boolean` | `false` | Activa el modo debug (bordes). |
| `fixed` | `boolean` | `false` | Indica si el elemento debe aparecer en todas las páginas. |
| `break` | `boolean` | `false` | Fuerza un salto de página antes del elemento. |

### Ejemplo

```tsx
<H1>Título</H1>
<H3>Subtítulo</H3>
<P>Parrafo con texto normal.</P>
```

## Strong, Em, U, Small

Componentes de formato de texto en línea. Soportan todas las propiedades del componente `Text`.

- **Strong**: Texto en negrita.
- **Em**: Texto en cursiva (énfasis).
- **U**: Texto subrayado.
- **Small**: Texto más pequeño.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales. |
| `debug` | `boolean` | `false` | Activa el modo debug (bordes). |
| `fixed` | `boolean` | `false` | Indica si el elemento debe aparecer en todas las páginas. |
| `break` | `boolean` | `false` | Fuerza un salto de página antes del elemento. |

### Ejemplo

```tsx
<P>
  <Strong>Negrita</Strong>, <Em>Cursiva</Em>, <U>Subrayado</U>, <Small>Pequeño</Small>
</P>
```
