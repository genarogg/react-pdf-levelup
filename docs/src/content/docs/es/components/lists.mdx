## UL

Lista desordenada (Unordered List).

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales. |
| `type` | `string` | `"disc"` | Tipo de viñeta (disc, circle, square). |

### Ejemplo

```tsx
<UL type="square">
  <LI>Elemento 1</LI>
  <LI>Elemento 2</LI>
</UL>
```

## OL

Lista ordenada (Ordered List).

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales. |
| `type` | `string` | `"decimal"` | Tipo de numeración (decimal, lower-alpha, upper-alpha, lower-roman, upper-roman). |
| `start` | `number` | `1` | Número inicial de la lista. |

### Ejemplo

```tsx
<OL type="upper-roman" start={3}>
  <LI>Item III</LI>
  <LI>Item IV</LI>
</OL>
```

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
