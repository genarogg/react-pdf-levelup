## Left

Alinea el contenido a la izquierda.

### Props

| Nombre | Tipo | Default | Descripción |
| :--- | :--- | :--- | :--- |
| `style` | `object` | `{}` | Estilos adicionales. |

### Ejemplo

```tsx
import { Left, P } from '@react-pdf-levelup/core';

<Left><P>Texto a la izquierda</P></Left>
```
