title: Integración Backend
description: Generación de PDFs en el servidor o scripts con Node.js.
---

React PDF Level Up no es solo para el frontend. Puedes utilizar la función `generatePDF` en entornos de Node.js para generar documentos programáticamente, guardarlos en disco o enviarlos como respuesta HTTP.

## Requisitos

Asegúrate de que tu entorno soporte React. Generalmente esto funciona out-of-the-box en frameworks como **Next.js**, **Remix**, **express**, **fastify**, o en scripts simples de Node, bun o cualquier entorno que soporte ES Modules.

## Ejemplo de Script

El siguiente ejemplo muestra cómo generar un PDF, convertirlo a Buffer y guardarlo en el sistema de archivos o enviarlo al frontend. 

```typescript
import { generatePDF } from "@react-pdf-levelup/core";
import Template from "./MyTemplate"; // Tu componente Template

const controllerX = async () => {
    try {
        const data = {
            nombre: "Juan Pérez",
            edad: 30,
            ciudad: "Madrid",
        };

        // 1. Generar el PDF en base64
        const base64PDF = await generatePDF({ template: Template, data });

        if (!base64PDF) {
            throw new Error("No se pudo generar el PDF");
        }

        return base64PDF;

    } catch (error) {
        console.error("Error generating PDF:", error);
    }
};

```

## Uso en API Endpoints (desde su backend).

Si estás utilizando Express, Next.js API Routes u otro servidor HTTP, puedes enviar el buffer directamente al cliente y decodificarlo en el frontend.

## No utilizo entorno de ejecucion javascript/typescript.

¿puedo usar React PDF Level Up? Si, puedes utilizar el api rest disenada para generar los pdfs, puedes encontrar la documentacion de esta api en [este enlace](https://react-pdf-levelup.nimbux.cloud/docs/es/guides/api-rest/)
