## generatePDF

Función asíncrona para generar un PDF en formato base64 a partir de un componente React.

### Firma

```typescript
generatePDF({ template, data }: { template: React.ElementType, data?: any }): Promise<string>
```

### Parámetros

| Nombre | Tipo | Descripción |
| :--- | :--- | :--- |
| `template` | `React.ElementType` | El componente React que representa el documento PDF. |
| `data` | `any` | Datos opcionales que se pasarán como props al componente template. |

### Retorno

Retorna una promesa que resuelve a un `string` conteniendo el PDF codificado en base64.

### Ejemplo

```tsx
import { generatePDF } from 'react-pdf-levelup';
import MyDocument from './MyDocument';

const handleGenerate = async () => {
  const base64 = await generatePDF({ 
    template: MyDocument, 
    data: { name: "John Doe" } 
  });
  console.log("PDF generado:", base64);
};
```

## decodeBase64Pdf

Función para decodificar un string base64 de PDF, descargarlo y abrirlo en una nueva pestaña. Funciona en el navegador.

### Firma

```typescript
decodeBase64Pdf(base64: string, fileName: string): void
```

### Parámetros

| Nombre | Tipo | Descripción |
| :--- | :--- | :--- |
| `base64` | `string` | El string base64 del PDF. |
| `fileName` | `string` | El nombre con el que se descargará el archivo. |

### Ejemplo

```tsx
import { decodeBase64Pdf } from 'react-pdf-levelup';

const handleDownload = () => {
  decodeBase64Pdf(base64String, "documento.pdf");
};
```
